# Panduan Kontribusi - NizamCraft

Kami sangat senang Anda tertarik untuk berkontribusi pada NizamCraft! Dengan berpartisipasi dalam proyek ini, Anda membantu membuatnya lebih baik untuk semua orang.

## Cara Berkontribusi

### 1. Melaporkan Masalah (Bug Reports)
Jika Anda menemukan bug atau masalah:
- Pastikan masalah tersebut belum dilaporkan sebelumnya di bagian Issues.
- Buat Issue baru dengan menjelaskan masalah secara rinci, langkah-langkah untuk mereproduksi bug, serta perilaku yang diharapkan.
- Lampirkan screenshot atau pesan error jika ada.

### 2. Mengusulkan Fitur Baru
Kami selalu terbuka untuk ide-ide baru!
- Buka Issue baru dan jelaskan fitur yang ingin Anda tambahkan secara konseptual.
- Jelaskan mengapa fitur tersebut berguna dan bagaimana cara kerjanya.

### 3. Mengirimkan Pull Request (PR)
Jika Anda ingin memperbaiki bug atau menambahkan fitur sendiri:
1. Fork repository ini.
2. Buat branch baru dari branch utama (`git checkout -b fitur-baru-anda`).
3. Lakukan perubahan kode Anda. Pastikan kode bersih, rapi, dan terdokumentasi.
4. Lakukan pengujian secara lokal untuk memastikan perubahan tidak merusak fitur lain.
5. Commit perubahan Anda dengan pesan commit yang deskriptif (`git commit -m "Menambahkan fitur X"`).
6. Push branch Anda ke fork Anda (`git push origin fitur-baru-anda`).
7. Buka Pull Request ke branch utama repository asli.

## Pedoman Pembuatan Kode

- **Gaya Penulisan**: Gunakan standar ES6 JavaScript yang bersih dan modern.
- **Komentar**: Berikan komentar pada bagian kode yang kompleks agar mudah dipahami oleh kontributor lain.
- **Kompabilitas**: Pastikan kode baru Anda tetap responsif di layar perangkat mobile dan tidak merusak kompatibilitas browser Safari iOS/MacBook.

Terima kasih atas kontribusi Anda! 🚀
